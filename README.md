# THE ULTIMATE ONSHAPER'S MACROPAD

My hackpad-inspired macropad for the Hack Club Macondo event! It's designed for ultimate CAD lock in sessions. It has 9 Cherry MX switches, 9 blank keycaps, and a Seeed XIAO RP2040. The PCB was created in KICAD and the case was designed on Onshape. 

The controls will be:

Row 1: Line (l), Point (shift + s), 3 Point arc (a)

Row 2: Center point circle (c), Center point rectangle (r), Corner rectangle (g)

Row 3: Construction (q), Mirror (m), Dimension (d)
